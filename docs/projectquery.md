# AI-Powered Restaurant Recommendation System (Zomato Use Case)

## Problem Statement
Build an AI-powered restaurant recommendation service inspired by Zomato. The system should combine structured restaurant data with a Large Language Model (LLM) to generate personalized recommendations.

## Objective
Design and implement an application that:
- Collects user preferences (location, budget, cuisine, minimum rating, optional notes)
- Uses a real-world restaurant dataset
- Generates ranked recommendations with an LLM
- Explains why each recommendation matches the user request

## Core Workflow
1. Data ingestion and normalization
2. User preference capture and validation
3. Candidate filtering and prompt preparation
4. LLM-based ranking and explanation
5. Result display in a clear user-facing format
